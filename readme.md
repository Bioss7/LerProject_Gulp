# Git ProjectGulp

Работа с Gulp используя консоль git bash
Необходимые установить:
Gitbash: https://git-scm.com/downloads
Node.js: https://nodejs.org/en/
Gulp: https://gulpjs.com/

Команды которые надо вводить в консоле git bash:
ncu -u --packageFile package.json // Проверка пакетов обновлений
npm i                             // Установка
gulp                              // Запуск и сборка проекта на локальном сервере
